# Permitful

Permitful provides an interactive map that allows users to visualize where all of the most recently updated building permits are in a specific jurisdiction.

## Technologies

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
